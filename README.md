# Health-monitoring-system
Health Monitoring is an IoT-based embedded project developed to monitor a patient's vital health parameters in real time. This  system uses sensors to measure blood pressure and heart rate. The collected data can be transmitted through IoT technology, allowing  remote monitoring by doctors or family members.
